# POC実装計画書

## 1. プロジェクト概要

### 1.1 背景
出光興産の建設現場において、工事発注時の価格折衝で想定工数と実績工数の乖離が問題となっている。
カメラによる作業員の自動計測システムを構築し、客観的な工数データを取得することで解決を目指す。

### 1.2 システム概要
- **カメラ**: Wi-Fi HaLow対応 FWCカメラ（静止画、1-60分間隔）
- **AI処理**: YOLO人物検出 + Re-ID（同一人物識別）
- **出力**: 日次の人数/時間グラフ、実人工算出

### 1.3 目標KPI
| 指標 | 目標値 |
|------|--------|
| 検出率（昼間） | >= 95% |
| 検出率（夜間） | >= 90% |
| 重複カウント率 | <= 5% |
| 実人工算出誤差 | +/- 10% |
